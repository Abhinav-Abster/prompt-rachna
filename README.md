## Usage

- Scroll down to progress through the story.
- Watch the particles shift colors and the background transition from dark to light.
- Pay attention to the "First Glimpse" section for a unique parallax effect.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, flexbox, grid, and animations.
- **JavaScript (ES6+)**: Core logic and DOM manipulation.
- **GSAP (GreenSock Animation Platform)**: Advanced animation library.
- **ScrollTrigger**: Scroll-based animation plugin for GSAP.
